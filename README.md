# pink

## Build Setup:

``` bash
# Download repository:
git clone https://github.com/RomanBykov/pink

# Go to the app:
cd pink

# Install dependencies:
npm install

# Server with hot reload at http://localhost:3000/
npm start

# Output will be at build
npm run build
```
